# Behance — Overview

## What Behance Represents

Behance is Adobe's global creative portfolio platform — the largest curated showcase of design work on the internet. It hosts work from hundreds of thousands of graphic designers, UI/UX designers, brand studios, illustrators, photographers, motion designers, and creative directors across every country. Projects range from student work to campaigns produced by the world's top agencies.

Within Holocraft's reference hierarchy, Behance represents the **UI/UX inspiration library** — the source for understanding how design professionals present creative work at its best. It is not a template source. It is a window into the visual language that defines "premium" in the design industry.

Specifically: when Holocraft generates an agency portfolio site, a design studio's homepage, a product showcase, or a creative's personal brand — the aesthetic authority comes from Behance. The work shown here sets the bar for what sophisticated clients expect.

## Core Philosophy

**Let the work speak.** The fundamental principle of Behance's design language — applied both to the platform's own UI and to the best projects it hosts — is that chrome recedes and content advances. Every navigation element, metadata tag, and UI affordance is designed to be invisible enough that the creative work dominates without competition.

**Minimal chrome, maximum content.** The platform's own interface is borderline invisible: white space, thin fonts, neutral greys. This is not poverty of design — it is discipline. The restraint of the container makes the content inside it appear more powerful.

**Editorial clarity.** The best Behance projects read like premium editorial layouts — magazine spreads translated to screen. Images are large, breathing room is generous, text is minimal and purposeful. The hierarchy is obvious at first glance and rewards closer reading.

**Curation signals quality.** Being featured on Behance's front page or curated galleries is a professional credential. The platform's aesthetic codes (whitespace, image quality, layout polish) function as trust signals for creative professionals.

## Industries and Use Cases

- **Design agencies** — showcasing client work, case studies, creative capabilities
- **Freelance designers** — personal portfolio, skills advertising, client acquisition
- **Product designers** — UX case studies with process documentation, before/after screens
- **Brand studios** — identity work, brand guidelines presentations, logo systems
- **Photographers** — photo essays, editorial work, commercial campaigns
- **Motion designers** — video showcases, GIF previews, frame sequences
- **Illustrators** — series work, style demonstration, commissioned pieces
- **Art directors** — campaign work, art direction documentation, set photography

## Emotional Direction

- **Curation** — the sense that every element shown has been selected deliberately. Nothing is accidental.
- **Craft** — precision in execution. Alignment is exact. Spacing is intentional. Color relationships are studied.
- **Professional pride** — the portfolio is a professional artifact, not just a website. It carries the weight of a career.
- **Creative aspiration** — viewing great Behance work creates the desire to produce work at that level. This aspirational tension is commercially valuable for portfolio clients.

## What Holocraft Extracts from Behance

### Case Study Layout Patterns
The anatomy of a premium design case study:
1. Full-bleed hero image or video (100vw, no sidebar)
2. Metadata row: client, role, year, tools used — tight, left-aligned
3. Brief/challenge section: short paragraphs, editorial line-height
4. Full-width process images (sketches, wireframes, iterations) — no captions by default
5. Split sections: image left + annotation right (or reversed)
6. Final outcome: large, proud, full-bleed final renders
7. Results/impact: clean stat callouts or paragraph

### Image Presentation Techniques
- **Full-bleed**: images extend to the viewport edge (`width: 100vw; margin-left: calc(-50vw + 50%)`) — the most powerful presentation technique
- **Device mockups**: work shown on realistic device frames (browser, iPhone, tablet). Creates context.
- **Flat lays**: physical materials photographed from above. Branding work especially.
- **Dark background presentation**: many designers present their work on `#1a1a1a` or `#0d0d0d` — makes colors pop and feels premium
- **Consistent aspect ratios**: 16:9 for screen work, 4:3 or 3:2 for print/physical work

### Editorial Grid Systems
The best Behance projects use deliberate grid decisions:
- **Full-bleed**: 100% width, no container
- **Constrained content**: 960px or 1200px max-width container for body text and small images
- **Asymmetric grids**: 60/40 or 70/30 splits for image+text sections
- **Breathing room**: 80-120px vertical gaps between major sections (not the 80px BootstrapMade standard — true breathing room at 120-160px for portfolio work)

### Typography Hierarchy for Showcasing Work
When text appears in a portfolio context, it functions as editorial support, not marketing copy. The hierarchy is:
- Project title: large, confident, minimal
- Metadata labels: small, uppercase, systematic
- Body/case study copy: generous line-height (1.8+), narrow column (max 680px), professional tone
- Image captions: small, light, subordinate

### Patterns to Study in Detail
- **Project grid**: hover states, masonry vs. uniform, thumbnail treatment
- **Full-bleed image sections**: how to implement `100vw` width images inside a constrained layout without horizontal overflow
- **Before/after layouts**: side-by-side or swipe comparisons for redesign showcases
- **Device mockup presentation**: how to stage work in context (perspective frames, flat displays)
- **Process documentation**: showing the messy middle (wireframes, sketches, iterations) as a quality signal

## What Behance Does NOT Do (and Why It Matters)

- **No excessive animation** — the work itself should be engaging, not the chrome around it
- **No dark mode by default** — the platform is predominantly white, letting images set the tone
- **No complex navigation** — simple global nav, project pages are linear scrolls
- **No sidebar cluttering the work** — full-width content first

For Holocraft: Behance teaches restraint. Understanding when NOT to add motion, color, or decoration is as important as knowing when to apply them. Agency portfolio sites routinely fail because they compete with the work instead of serving it.
